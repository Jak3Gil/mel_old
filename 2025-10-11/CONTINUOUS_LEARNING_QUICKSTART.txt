🚀 START LEARNING (ONE COMMAND)

  ./scripts/run_continuous_learning.sh

  What it does:
  ✓ Creates necessary directories
  ✓ Builds the watcher
  ✓ Starts continuous learning loop

📊 WATCH IT LEARN

  # Monitor metrics (live)
  tail -f logs/continuous_learning_metrics.csv

  # Check processed lessons
  ls data/processed/

  # Check for failures
  ls data/failed/

  # View latest snapshot
  ls -lht data/snapshots/ | head -2

⚙️ CONFIGURATION

  Edit: config/continuous_learning.yaml

  Key settings:
  - inbox_dir: Where to watch for .tch files
  - poll_seconds: How often to check (default: 3)
  - snapshot_every_seconds: Snapshot interval (default: 60)
  - metrics_every_seconds: Metrics logging (default: 10)

🛑 STOP LEARNING

  Press Ctrl+C in the terminal running the watcher

  All progress is automatically saved in:
  - data/melvin_brain.bin (main brain state)
  - data/snapshots/ (timestamped backups)
  - logs/continuous_learning_metrics.csv (metrics)

📁 KEY DIRECTORIES

  data/inbox/       ← DROP .tch FILES HERE
  data/processed/   ← Successfully learned lessons
  data/failed/      ← Failed lessons (check for errors)
  data/feeds/       ← Sample lessons to copy from
  data/snapshots/   ← Periodic brain backups
  logs/             ← Metrics CSV
